## Key Technical Information

**Frontend Stack:** React 18.3.1, Vite 6.3.5, TypeScript, Tailwind CSS, Radix UI
**Backend API:** FastAPI with POST /api/v1/analyze endpoint
**Architecture:** 5-layer system (UI, Application, Processing Engine, AI Models, Data)
**Port:** Frontend runs on localhost:3000
